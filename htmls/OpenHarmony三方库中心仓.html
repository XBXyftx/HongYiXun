介绍

circle_navigation一个支持多方位置的圆形导航菜单，支持自定义组件，支持动态设置菜单按钮，支持左上右下，左上、右上，左下、右下，八个方位弹出，满足不同场景化需求。

支持Api版本

Api适用版本：>=12

目前功能

  • 1、支持自定义组件，可按照自己要求进行绘制。
  • 2、支持动态设置菜单也就是导航个数。
  • 3、支持左上右下等多方位置弹出。

效果

目前支持多个位置弹出，案例效果仅列出几个，更多效果请查看Demo。

快速使用

方式一：在Terminal窗口中，执行如下命令安装三方包，DevEco Studio会自动在工程的oh-package.json5中自动添加三方包依赖。

建议：在使用的模块路径下进行执行命令。

ohpm install @abner/circle_navigation

方式二：在工程的oh-package.json5中设置三方包依赖，配置示例如下：

"dependencies": { "@abner/circle_navigation": "^1.0.1"}

代码使用

需要实现三个属性，parentLayout属性是，父布局，也就是触发的视图，childLayout是子布局，是圆形导航菜单，clickStatus是展示状态。

CircleNavigation({
        clickStatus: this.clickStatus,
        parentLayout: () => {
          this.parentLayout()
        },
        childLayout: (position: number) => {
          this.childLayout(position)
        }
      })

完整案例

@Entry
@ComponentV2
struct FullPage {
  @Local clickStatus: boolean = false

  @Builder
  parentLayout() {
    Column() {

    }
    .width(100)
    .height(100)
    .borderRadius(100)
    .backgroundColor(Color.Blue)
    .onClick(() => {
      //点击
      this.clickStatus = !this.clickStatus
    })
  }

  @Builder
  childLayout(position: number) {
    Column() {
      Text((position + 1).toString())
        .fontColor(Color.White)
        .fontWeight(FontWeight.Bold)
    }
    .justifyContent(FlexAlign.Center)
    .width(50)
    .height(50)
    .borderRadius(50)
    .backgroundColor(Color.Blue)
  }

  build() {
    RelativeContainer() {
      ActionBar({ title: "全圆" })
        .alignRules({
          middle: { anchor: '__container__', align: HorizontalAlign.Center }
        })
      CircleNavigation({
        clickStatus: this.clickStatus,
        parentLayout: () => {
          this.parentLayout()
        },
        childLayout: (position: number) => {
          this.childLayout(position)
        }
      })
        .alignRules({
          center: { anchor: '__container__', align: VerticalAlign.Center },
          middle: { anchor: '__container__', align: HorizontalAlign.Center }
        })
    }
    .height('100%')
    .width('100%')
  }
}

相关属性

属性 类型 概述
parentLayout @BuilderParam 父布局，触发圆形导航
childLayout @BuilderParam 子布局，展示圆形导航
clickStatus boolean 点击状态，用于显示圆形导航菜单
quantity number 圆形导航菜单个数
radius number 扩散半径参数，圆形导航菜单和父布局的距离
circleType CircleType 展示位置，默认FULL是全圆，详细可见下面介绍
circleOffset number 偏移量，导航菜单偏移量，比如0.1，0.2这种，便于导航菜单聚集
duration number 导航菜单时间，默认500毫秒

CircleType

圆形导航菜单位置

参数 概述
FULL 全圆
LEFT_HALF 左半边
RIGHT_HALF 右半边
TOP_HALF 上半边
BOTTOM_HALF 下半边
TOP_LEFT_QUARTER 左上四分之一
TOP_RIGHT_QUARTER 右上四分之一
BOTTOM_LEFT_QUARTER 左下四分之一
BOTTOM_RIGHT_QUARTER 右下四分之一
